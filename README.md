# Mood Distribution Data Analysis

## Overview
This project performs exploratory data analysis on a mood tracking dataset to understand emotional distribution patterns.

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn
- NumPy

## Analysis Performed
- Mood frequency distribution
- Statistical summary
- Visualization of emotional trends
- Identification of dominant emotional states

## Key Findings
- Neutral and Happy moods dominate dataset
- Negative emotions show lower frequency
- Emotional distribution suggests skew towards positive states

## Project Structure
- Notebook with complete EDA workflow
- Cleaned dataset
- Visualization outputs
